Update the realm-export.json file#

Gather required values#

Get the values needed to update the realm-export.json file.

Value

Description

<hostname>

Host name of the VM identified in the Gather required values section of Update the .env file.

<domain_suffix>

Full domain suffix of the VM identified in the Gather required values section of Update the .env file.

<domain_short_suffix>

Short domain suffix of the VM identified in the Gather required values section of Update the .env file.

<TR_PORT>

Port that is set in the Edit environment variables section of Update the .env file.

Update JSON key entries#

  1. In Ansys_App_Portal_and_Deployment_Template, go to the src/ansys/saf/one_node/portal-packaged/keycloak directory.

  2. Open realm-export.json.

  3. Navigate to the section where the following value is set:

    realm-export.json#
    "clientId": "oauth2-proxy"

  4. In the same section, find the following key entries:

    • "rootUrl"

    • "adminUrl"

    • "baseUrl"

    • "redirectUris"

  5. Update these key entries with the following values:

    realm-export.json#
    "rootUrl": "http://portal.<hostname>.<domain_suffix>:<TR_PORT>"

"adminUrl": "http://portal.<hostname>.<domain_suffix>:<TR_PORT>"

"baseUrl": "http://portal.<hostname>.<domain_suffix>:<TR_PORT>/"

"redirectUris": [
"http://oauth.<hostname>.<domain_suffix>:<TR_PORT>/oauth2/callback"
]
