Update the docker-compose.yaml file#

  1. Open the docker-compose.yaml file at the root of the Ansys_App_Portal_and_Deployment_Template directory.

  2. Update the following labels:

    1. In the oauth service under labels, update traefik.http.routers.oauth.rule:

      docker-compose.yaml#
      "traefik.http.routers.oauth.rule": "(HostRegexp(`{subdomains:.*}.${OP_BASE_DOMAIN}`) || Host(`${OP_BASE_DOMAIN}`)) && (PathPrefix(`/oauth2`) || PathPrefix(`/saf/portal/oauth2`) || PathPrefix(`/saf/portal/api/oauth2`))"

    2. In the traefik service under labels, update traefik.http.routers.dashboard.rule:

      docker-compose.yaml#
      "traefik.http.routers.dashboard.rule": "Host(`TR_HOST_URL`)"

  3. In a WSL terminal, fetch the value of docker IP:

    • WSL
    ip addr show docker0

  4. In docker-compose.yaml, add an extra_hosts section with the value of docker IP to the following services:

    1. Add extra_hosts to the oauth service:

      docker-compose.yaml#
             extra_hosts:
         - ${KC_DOMAIN}=<docker IP>

    2. Add extra_hosts to the server service:

      docker-compose.yaml#
             extra_hosts:
         - ${KC_DOMAIN}=<docker IP>
