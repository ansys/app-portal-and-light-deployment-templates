Uninstalling the platform
#########################

To uninstall the app platform, follow the next steps:

#. Remove all existing containers:

   .. tabs::

     .. code-tab:: bash
         :caption: WSL

         docker rm -f $(docker ps -a -q)

#. Remove all existing volumes:

   .. tabs::

     .. code-tab:: bash
         :caption: WSL

         docker volume rm $(docker volume ls -q)

#. Remove all existing images:

   .. tabs::

     .. code-tab:: bash
         :caption: WSL

         docker rmi -f $(docker images -q)

#. Remove all existing networks:

   .. tabs::

     .. code-tab:: bash
         :caption: WSL

         docker network rm $(docker network ls -q | grep -vE '^(bridge|host|none)$')

#. Remove all cache files:

   .. tabs::

     .. code-tab:: bash
         :caption: WSL

         docker system prune -a
         docker volume prune -a

#. Ensure that docker is clean:

   .. tabs::

     .. code-tab:: bash
         :caption: WSL

         docker ps -a && docker images && docker volume ls

   Once above command is executed, you should see the following output:

   .. image:: /_static/documenting/docker-clean.png
        :width: 60%


#. Delete the content of the Solutions directory path (default to ``C:\Users\<user_name>\AppData\Roaming\Ansys\solutions``)
